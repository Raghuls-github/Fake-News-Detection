# Fake-News-Detection
This program can find the percentage of fake and real news given in a dataset. 
The packages and libraries used for this project are as follows,
 - nltk
 - Pandas
 - Numpy
 - Matplotlib
 - Seaborn
 - sklearn
 - Wordcloud

</B> Methods </B>
 - Logistic Regression
 - Decision Tree Classifier
 - Random Forest Classifier
